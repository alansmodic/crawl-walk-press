# Crawl, Walk, Press

**30 days to master the WordPress AI stack.**

A free, open-source training program for editorial and product teams. Learn to use the WordPress AI Experiments plugin, WordPress MCP, Studio CLI & Code, Telex, WordPress Playground, and the Abilities API — from zero to building real editorial workflows.

[**▶ Launch in WordPress Playground**](https://playground.wordpress.net/?blueprint-url=https://raw.githubusercontent.com/alansmodic/crawl-walk-press/main/blueprint.json) — try the full course in your browser, no install needed

## Get Started

Pick the option that works for you:

### Option 1: Try it instantly in Playground (no install)

Click the link above — the full 35-page course loads in your browser via WordPress Playground. Nothing to install, nothing to configure. Great for previewing or following along on any device.

### Option 2: Run it locally with Studio CLI

```bash
brew install --cask studio
git clone https://github.com/alansmodic/crawl-walk-press.git ~/Studio/crawl-walk-press
```

Open Studio, add the site, and start it. The SQLite database is included — all 35 pages are ready. This gives you a persistent local site with full WP-CLI access for the hands-on exercises.

### Option 3: Read the markdown (offline / print-friendly)

Download [**crawl-walk-press.md**](crawl-walk-press.md) — the entire course as a single structured markdown file with table of contents. Works in any markdown reader, or print it.

### Option 4: Track progress in Todoist

Import [**todoist-template.csv**](todoist-template.csv) into Todoist as a new project. All 29 tasks organized by week with descriptions and time estimates. Go to Todoist → Project → Import from CSV.

## What's Included

- **35 pages** of structured training content (Days 0–30, organized by week)
- **Custom child theme** (`cw5k`) based on Twenty Twenty-Five with polished design tokens
- **Progress tracker plugin** — localStorage-based, zero-friction completion tracking
- **SQLite database** — full site state, no import needed
- **XML content export** — human-readable content for diffing changes
- **Playground blueprint** — one-click launch in the browser
- **Markdown export** — full course as a single readable file
- **Todoist template** — importable task list for tracking progress

## The Stack

| Tool | What It Does |
|------|-------------|
| WordPress AI Experiments Plugin | Title gen, excerpts, alt text, content optimization |
| WordPress MCP | Connect Claude and AI agents to your WordPress site |
| Studio CLI & Studio Code | Local dev + AI-powered theme/site building |
| Telex | Describe an idea → get a WordPress block or theme |
| WordPress Playground | Browser-based WordPress sandbox, zero install |
| Abilities API | WordPress core framework for AI-discoverable capabilities |
| WordPress 7.0 AI Client | Provider-agnostic AI SDK in WordPress core |
